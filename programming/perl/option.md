
单行命令部分参数
------------------------------------

1. -p 可以自动生成一段小程序如下
  ```perl
  while (<>) {
      # code
      print;
  }
  ```
2. -n 和-p差不多，去掉了print语句
3. -i 如同在程序开头写上 `$^I=argv` ，例如`-i.bak`，将原文件备份为该后缀；如果不想备份，直接写上-i
4. -e 后面跟着的是可执行的程序代码
5. -a 自动分割（也意味着-n）
6. -l 自动在输出添加换行符
7. -c 检查语法但不执行程序
8. -d 进入调试模式
9. -w 打开警告

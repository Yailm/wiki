
sort
------------------------------------

### 参数说明

* -n 按数字进行排序
* -d 按字典序进行排序 （这两个都考虑第一个字符）
* -r 逆序
* -k N 指定按第N列排序
* -b 忽略前导空白
* -f 将小写都转换成大写
* -g 按数字大小进行排序（考虑整个列）
* -M 按月份进行排序 (unknown) 'JAN' .. 'DEC'
* -i 只对可打印字符进行排序
* -h 人类可阅读的数字排序 (e.g. 2K 1G)
* -R 随机排序
* -V 版本号排序
* -o FILE 指定输出文件
* -t SEP 指定列的分隔符

### 示例

    sort -nrk 1 data.txt
    sort -bd data   // 忽略像空格之类的前导空白字符
